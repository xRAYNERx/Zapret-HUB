# Changelog

## [1.0.5] — 2026-06-17

- Уведомления отображаются в правом верхнем углу с улучшенной читаемостью
- Кнопка «Копировать» для адреса TG Proxy на главной
- Тумблер «Сворачивать при старте» — окно сразу уходит в трей при запуске
- В окне «Обход остановился» добавлена кнопка «Подобрать стратегию»
- Иконка TgWsProxy скрывается из области уведомлений Windows

## [1.0.4] — 2026-06-17

- Единый поток обновления TG Proxy на главной и в разделе «Сервис»
- Общая логика `isUpdateAvailable()` — статусы «есть обновление» / «актуально» больше не расходятся
- После обновления TG в «Сервисе» главная сразу перечитывает статус (и наоборот)
- Прогресс-бар при обновлении TG Proxy в разделе «Сервис»

## [1.0.3] — 2026-06-17

- Обновление Zapret HUB **внутри приложения**: кнопка «Обновить» скачивает установщик и ставит тихо (`/S`), без браузера
- Прогресс-бар при ручном обновлении HUB и при «Обновить всё» при старте
- Исправлена проверка обновлений при лимите GitHub API (fallback на страницу релиза)

## [1.0.2] — 2026-06-17

- Онбординг: пошаговый пропуск, крестик закрытия, переход к следующему шагу после подбора стратегии
- Убраны описания конфигов на главном экране
- Единая типографика в онбординге и плашках рекомендаций
- Сервис: разделитель и тумблер «Проверять обновления при запуске» рядом с текстом

## [1.0.1] — 2026-06-17

- Автоматическая проверка обновлений при запуске (Zapret HUB, движок обхода, TG Proxy)
- Окно с предложением обновить доступные компоненты
- Обновление Zapret HUB через скачивание установщика с GitHub
- Подбор рабочей стратегии — проверка всех стратегий или одной на вашем ПК
- Кнопка «Проверить все» для обновлений в разделе «Обслуживание»
- Отображение версий движка и TG Proxy на главном экране

## [1.0.0] — 2026-06-16

Первый релиз Zapret HUB.

- Обход Discord, YouTube и других сайтов
- Прокси Telegram
- Автозапуск Zapret и Telegram по отдельности
- Свои списки сайтов
- Установщик и portable-версия
